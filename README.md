# RM-campus-visual-recognition
### 华南理工大学 2021机器人对抗赛 正赛视觉部分 识别几何图案

#### 主要思路
主要的思路是先进行灰度处理和两次二值化，然后用canny边缘检测算法提取边缘，用中值滤波和高斯模糊排除干扰，并对边缘进行膨胀处理避免重复计算，最后用findcontours函数查找边缘，用多边形逼近并通过边数判断形状，大于15条边则认为是圆形。  

#### 一些小小的创新点和发现
程序中生成了多个轨迹条，可以有效地针对现场情况进行调整。例如，二值化默认为关闭，若反光较为严重（尤其是镜面反射）时，或识别出的图形反复横跳时，可以抬高“二值图 1”的阈值，滤除低频无关信号。若摄像头视野较为杂乱，则可以降低或是关闭二值化，调整canny阈值，避免使原本平滑的表面出现边缘。具体效果可以参考同时上传的测试截图。

程序中还对轮廓的边长和面积进行限制，面积可以通过轨迹条进行调整，以降低误判概率。同时通过实验发现，多边形逼近对图形的误判绝大多数情况下只会增加边数，例如可能将四边形识别为五边形，而不会识别为三角形，故在判断图形时直接由小到大判断边数，不为零的即为所寻找的图形。此方法也可以一定程度上降低误判的概率。
